# Project template

Execute:

`npx create-next-app@latest -e https://github.com/carlesandres/project-template`

You **don't** need to create the project subfolder before executing the command.

This project template includes:

- Next.js
- Typescript
- Tailwind
- Eslint
- Stylelint
- Husky
- shadcn/ui
- zod
- lucide
